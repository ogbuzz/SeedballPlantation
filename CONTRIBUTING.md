# Comment contribuer à SeedballPlantation

Merci de ton intérêt pour ce projet de régénération des sols via seedballs !

## Autres Façons de contribuer
- **Contenu sur SeedballPlantation.com** : Ajoute ou améliore une recette de seedballs testée terrain (ratios, climats, résultats).
- **Traductions** : FR/EN déjà là ; ajoute ES, PT, SW, AR pour toucher l'Afrique/Latam.
- **Code** : Améliore le frontend Vanilla JS, ajoute features (ex. : filtres par sol/climat).
- **Docs** : Complète la doc v3 ou ajoute tutoriels VPS.
- **Design** : Icônes, photos terrain, ou UX mobile.

## Étapes pour contribuer
1. Fork ce repo.
2. Crée une branche : `git checkout -b feat/ma-recette-desert`.
3. Commit : `git add . && git commit -m "Ajoute recette seedballs sahara"`.
4. Push : `git push origin feat/ma-recette-desert`.
5. Ouvre une Pull Request (décris ce que tu ajoutes).

On accepte tout en FR/EN/ES/PT/SW. Pas de drama, juste du positif pour la planète !

Questions ? Ouvre une Issue.
